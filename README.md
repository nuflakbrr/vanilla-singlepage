# github.com/nuflakbrr/vanilla-singlepage

Create Vanilla Single Page.

## Getting Started

1. Download this repository or use git clone this repository
2. Extract file.zip
3. Open Your Code Editor
4. Open Terminal
5. Run the development server:

```bash
node server.js
```
